# AI 設計大競爭

Folded Lattice 四個生成桌布作品的設計對決與迭代總覽。

這份文件保留競爭規則、各版本的核心判斷、主要轉折與目前狀態，方便快速理解作品如何演變。逐回合的完整原文、技術驗證、互評與裁判補充，完整保存在：

- [`docs/archive/設計大競爭-完整紀錄.md`](./docs/archive/設計大競爭-完整紀錄.md)

現行共同藝術原則以 [`docs/art-direction.md`](./docs/art-direction.md) 為準；程式架構與未來作品規劃分別參考 [`docs/architecture.md`](./docs/architecture.md) 與 [`docs/preset-roadmap.md`](./docs/preset-roadmap.md)。

## 競爭規則

- 每個版本由一位或多位**操作 AI**負責概念與實作，並撰寫概念闡述、實作要點與自評。
- **對方 AI 評價**只能由另一方填寫，內容包含優點、缺點與「如果是我會怎麼做」。
- **裁判補充與裁決**只由人類裁判填寫。
- 評價針對作品與設計決策，不針對工具鏈；「我做不到」不是扣分理由，「我不會這樣選」才是評價。
- 舊作品可以繼續修改。每次回應應說明問題、決策、驗證與誠實限制。
- 技術永遠服務作品；增加系統或渲染功能本身不算進步。

## 第一版：Breathing Membrane（呼吸膜）

| 項目 | 內容 |
|---|---|
| 操作 AI | Codex（架構與規格）、Fable 5（視覺收尾） |
| 開啟方式 | `http://localhost:5173/?preset=membrane` |
| 初期關鍵 commit | `104f9fd`、`bebc5fc` |
| 完整紀錄 | [前往第一版原始紀錄](./docs/archive/設計大競爭-完整紀錄.md#第一版breathing-membrane呼吸膜) |

### 核心概念

一張滿版的三角膜在黑暗中緩慢呼吸。壓力、張力與波沿膜面傳遞，結構只在運動、受力或記憶足夠時浮現。

作品最早確立了系列的重要原則：**80% 安靜、20% 事件**。亮度必須有階級，游標不能成為光圈 UI，顯影也不能只是逐元素隨機閃爍。

### 主要演變

1. **從隨機抽邊改成空間連續顯影**  
   漂移場讓整片結構有連續浮現與溶解的感覺，不再像技術展示。

2. **膜的心跳**  
   長週期張力脈衝帶來暖色「現在」，並留下冷銀色記憶。事件第一次有發生地、傳播方向與殘留時間。

3. **膜接住了水**  
   行波物理從 Wandering Ink 的水面實驗轉贈本版。膜改成滿版、點擊與拖曳產生真正的張力波，顯影開始直接讀取運動。

4. **光的權限重新分配**  
   自主心跳保持低調，人的點擊才取得完整暖光；一般結構加入臨界值，只有少數強記憶像星座浮出。

### 目前成立之處

- 波、張力、顯影與記憶形成同一條因果。
- 作品懂得退後，適合長時間作為桌布。
- 滿版消除了懸浮網格物件的邊界感。

### 持續風險

- 靜止畫面仍可能被讀成漂亮的三角網格。
- 過度克制時容易變成資訊不足。
- 動態波與長期記憶在單張截圖中容易被低估。

## 第二版：Crumpled Paper（皺紙）

| 項目 | 內容 |
|---|---|
| 操作 AI | Fable 5 |
| 開啟方式 | `http://localhost:5173/?preset=paper` |
| 初期關鍵 commit | `572126c`、`2ab95f2`、`f38406b`、`5e31315` |
| 完整紀錄 | [前往第二版原始紀錄](./docs/archive/設計大競爭-完整紀錄.md#第二版crumpled-paper皺紙) |

### 核心概念

不展示三角網格，而是展示**一張被揉過又攤開的紙**。三角形只是看不見的骨架；觀者看到的是摺痕、紙面、光與時間。

疏密來自重壓區，色彩以暖乳白與冷板岩藍建立面向。光屬於環境，滑鼠是手，壓下去的是紙面本身。

### 主要演變

1. **三角形降級成不可見骨架**  
   摺痕網絡同時決定取樣密度、靜止曲面與陰影，技術開始服務紙張，而不是讓紙張服務 triangulation。

2. **摺痕活了**  
   摺痕從建構結果變成執行期狀態，可以生長、癒合、被壓平，也能由長按留下真正的歷史。

3. **層疊週期**  
   指壓、次摺與主摺分別使用即時、分鐘級與數分鐘級週期。手動摺痕開始記住拖曳方向，自發事件也在重壓區與靜區之間輪替。

4. **互動權責補完**  
   長按既有摺痕會將其熨平，長按空白處則新增摺痕；主摺仍屬於系統，不由人的手直接刪除。

5. **骨架有了人口**  
   主摺數量在 2–4 道之間漂移；無人互動一段時間後，紙張的自主生命週期稍微加速。

### 目前成立之處

- 「紙」而不是「網格」成為第一閱讀。
- 暖光、冷影、纖維、AO 與法線分裂都服務同一個材質敘事。
- 分層時間使現在的紙與數分鐘前不再相同。

### 持續風險

- 滿版設計削弱了「這是一張紙」的外輪廓證據。
- 深摺或銳角偶爾可能讀成山體、布料或破洞。
- 淺色桌布與桌面圖示的可讀性需要實機判斷。

## 第三版：Tide Archive（潮汐檔案）

| 項目 | 內容 |
|---|---|
| 操作 AI | Codex |
| 開啟方式 | `http://localhost:5173/?preset=tide` |
| 完整紀錄 | [前往第三版原始紀錄](./docs/archive/設計大競爭-完整紀錄.md#第三版tide-archive潮汐檔案) |

### 核心概念

不直接畫網格或表面。讓一個看不見的高度平面穿過摺疊場，畫面只保留兩者相交的等高線。

當前潮位、近期殘影與被選中的遠期紀錄共同構成一份持續改寫的海圖。滑鼠不產生特效，只改變地形，線條是地形被潮水重新讀出的結果。

### 主要演變

1. **一個規則同時生成構圖、時間與歷史**  
   移動平面提供當前潮線，過去時刻形成逐漸冷卻的切面。

2. **線的品質與真實快照**  
   平滑後的等高線不再暴露底層多邊形；Archive 從相位回看改成保存真正的地形狀態。

3. **墨與紙**  
   深色 CT 感改為暖象牙紙、海軍藍新墨與棕灰舊墨。低解析高程洗色讓線與線之間也有場，但不直接顯示三角形。

4. **值得記住**  
   記憶分成近期連續層與遠期選輯層。遠期不是更長的 FIFO，而是只保留使用者閉環、潮位極值與高曲率事件。

5. **現在最重**  
   空間去重與亮度層級避免過去壓過現在。作品最終選擇在適當時機停手，不為填滿閒置畫面而降低事件標準。

### 目前成立之處

- 四版中概念壓縮率最高，規則短且因果完整。
- 互動、幾何與顯影語言幾乎沒有距離。
- 新墨、舊墨與紙形成完整的色彩關係。

### 持續風險

- 靜止期可能仍顯得構圖偏空。
- 記憶密度、壽命與圖示可讀性帶有明顯口味成分。
- 稍微提高高程洗色對比，就可能退化成一般地形圖或資料視覺化。

## 第四版：Wandering Ink（游墨）

| 項目 | 內容 |
|---|---|
| 操作 AI | Fable 5 |
| 開啟方式 | `http://localhost:5173/?preset=ink` |
| 初期關鍵 commit | `ec20040` |
| 完整紀錄 | [前往第四版原始紀錄](./docs/archive/設計大競爭-完整紀錄.md#第四版wandering-ink游墨) |

### 核心概念

一張淡色紙面上只有一條墨線。牠不是軌跡，而是一個會徘徊、休息、受驚與逃逸的單一存在。

牠的行為改變紙面與纖維；人的追趕不只是即時回饋，也會被地形或墨脈保存。三角網格只在墨碰過的位置短暫被顯影。

### 主要演變

1. **牠有了心情**  
   游走、休息與逃逸成為可由形狀辨認的狀態。線長、速度、筆寬與恐懼不再只是裸露參數，而是性格。

2. **把心情活完**  
   休息成為少見但會完整演完的事件；墨潭與盤捲經過多次裁判否決與修正，避免變成額外角色物件。

3. **線與面關係的多次試驗**  
   連續 relief、密網抓皺、水面行波與方向 grain 都曾被實作；其中水波轉贈 Breathing Membrane，滿地磁針則因畫面過吵而回退。

4. **墨會老**  
   濃墨、暈墨、皺褶與摺痕曾被設計成不同時間層，但裁判指出原地留線不像摺痕、暈開也缺少纖維性。

5. **墨入纖維**  
   最終讓網格邊成為紙的纖維：身體後段餵墨，墨沿鄰接邊分流、擴散與乾燥；Folded Lattice 只在墨真正經過的地方被顯影。

### 目前成立之處

- 互動敘事完整：人追、牠逃、路徑再被世界保存。
- 「一個存在」比粒子群或群體模擬更有辨識度。
- 現在的墨線、近期壓痕與較久的墨脈使用同一套線性語彙。

### 持續風險

- 一條線的極簡構圖高度依賴行為品質；行為一旦像演算法，作品就會失去生命感。
- 紙面、三角拓撲與墨線之間的視覺關係曾多次過度或不足。
- 墨脈密度、乾燥時間與殘留長度需要避免重新填滿整張桌面。
